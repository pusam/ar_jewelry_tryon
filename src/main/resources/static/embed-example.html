외부 쇼핑몰에서 AR 착용 연동하기

아래 코드를 쇼핑몰 상품 페이지에 삽입하면, 고객이 "착용하러가기"를 눌러 AR 가상 착용을 체험할 수 있습니다.

1. 간단한 링크 방식

<!-- AR_SERVER_URL: AR 서버 주소, JEWELRY_ID: 상품 ID -->
<a href="https://your-ar-server.com/tryon/1"
   class="ar-tryon-btn">
   착용하러가기
</a>

2. 스마트 배너 + 앱 설치 유도 방식 (권장)

<script>
(function() {
  const AR_SERVER = 'https://your-ar-server.com';

  document.querySelectorAll('.ar-tryon-btn').forEach(btn => {
    btn.addEventListener('click', function(e) {
      e.preventDefault();
      const jewelryId = this.dataset.jewelryId;
      const tryonUrl = AR_SERVER + '/tryon/' + jewelryId;

      // 1) PWA가 설치되어 있으면 바로 열기 시도
      // 2) 아니면 브라우저에서 AR 페이지 열기
      //    (AR 페이지 자체에서 PWA 설치를 유도함)
      window.open(tryonUrl, '_blank');
    });
  });
})();
</script>

<!-- 상품 페이지에 버튼 추가 -->
<a href="#" class="ar-tryon-btn" data-jewelry-id="1">
  📸 AR 착용하러가기
</a>

3. 동작 흐름

  1. 고객이 쇼핑몰에서 "착용하러가기" 클릭
  2. AR 서버의 착용 페이지로 이동
  3. PWA 미설치 시: 하단에 "앱 설치" 배너 표시 → 설치하면 홈 화면에 추가됨
  4. PWA 설치 후: 다음부터는 앱처럼 바로 실행
  5. 카메라가 켜지고 AI가 손/상체를 인식 → 주얼리를 실시간 오버레이
  6. 마음에 들면 촬영 후 쇼핑몰로 돌아가 구매
